Comandos básicos do Linux

whoami - para saber o usuário logado no momento

~ $ : significa que estamos rodando o sistema no modo de usuário normal
# : significa que estamos rodando o sistema no modo de usuário root

clear - limpar tudo na tela (control l)

ls - listar tudo que está dentro do diretório
ls -l - mostra a estrutura de permissões
ls -A: mostra os arquivos ocultos
cd - acessar pasta ( cd /home) (cd .. - retorna ao diretório anterior)

mkdir - criar diretório (mkdir ‘dioniluxcursodeterminal’)
rmdir ( remover diretório) rmdir ‘dioniluxcursodeterminal’)

pwd - dizer onde você está
touch - criar arquivo

rm -r - apagar pasta completa

sudo apt update - Atualizar lista de repositório do sistema

man - manual do comando ( man apt) …

list - listar pacotes baseados em algum nome ( apt list inkscape)

search - pesquisar nas descrições dos pacotes

sudo apt remove ( sudo apt remove scribus) - removendo o scribus
